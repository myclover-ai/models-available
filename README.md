# Clover Models Available

Model catalog for the [Clover](https://github.com/cloverly-ai/Clover) desktop AI app.

## Files

- `latest_clover_model.json` — The default recommended model for new installs (used by onboarding)
- `model_catalog_v2.json` — Full catalog of available models for the model hub

## Format

Each model entry follows the `CatalogModel` schema with quantization variants pointing to HuggingFace GGUF files.
